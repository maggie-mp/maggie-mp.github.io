    • Accessibility Statement:

    This website aims to follow WCAG 2.2 AA. I used Venngage and Colour Contrast Checker websites to make sure it met the contrast standard. Usability testing was done after the development of the website with a variety of people, including one tester with disabilities. Sadly, no assistive technology was used during the testing phase. This website was built with HTML, CSS and Javascript. The limitations of this website include: not all images include alternative text, sections are not in plain language, and this website may break on mobile devices.
    If you have any accessibility issues with this website, please contact me via email: magsmcperry@gmail.com.
